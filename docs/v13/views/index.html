GOV.UK Prototype Kit

Use the GOV.UK Prototype Kit to build realistic prototypes of your service that look and behave like GOV.UK.

The kit is used by multidisciplinary teams across government to design services. Anyone can use the kit, including:

  • content, interaction and service designers
  • developers
  • user researchers
  • product owners

To use the kit you'll need some basic knowledge of HTML, for example what tags and attributes are and how to copy and paste code. You can learn basic HTML and CSS on codecademy at https://www.codecademy.com.

The GOV.UK Prototype Kit provides a simple way to make interactive prototypes that look like services on GOV.UK. You can use these prototypes to get valuable feedback and insights, either from the people you work with or through user research.

You can copy example code from the GOV.UK Design System to add styles, components and patterns into your prototype. This means you can:

  • create prototypes quickly and easily
  • learn from the research and experience of other service teams
  • avoid repeating work that's already been done

Teams across government work in the open. If you cannot find what you need on the Design System, you can:

  • check the discussion around styles, components and patterns on Github
  • add an issue to the Prototype Kit backlog
